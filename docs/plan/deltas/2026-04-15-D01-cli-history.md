# D01 CLI History UX

## 변경 요약
- 인터랙티브 모드에 `↑/↓` 명령 히스토리 탐색 기능을 추가하는 확장 요구를 반영한다.
- 초기 계획 문서는 유지하고, 본 Delta로 변경 이력을 관리한다.

## 수용 기준
1. `↑`로 이전 명령, `↓`로 다음 명령 탐색 가능
2. 빈 입력/연속 중복 명령은 히스토리 저장 제외
3. 기존 `exit/quit`, 에러 후 재입력, 세미콜론 실행 흐름 유지

## 검증 계획
1. 명령 2개 입력 후 `↑`, `↑`, `↓` 순차 탐색 동작 확인
2. 오타 SQL 에러 후 `↑`로 직전 명령 복원 확인
3. 기존 기능 회귀 테스트 통과 확인

## 운영 메모
- 이후 유사 변경은 `D02`, `D03`...로 같은 형식으로 누적한다.
